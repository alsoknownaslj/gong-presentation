# gong-presentation

React + Vite app for the Gong presentation.

## Prerequisites

- **Node.js** 18 or newer (includes `npm`)

Check versions:

```bash
node -v
npm -v
```

## Run locally

From the project root:

```bash
npm install
npm run dev
```

Vite prints a local URL (usually `http://localhost:5173`). Open it in your browser. The dev server reloads when you change files.

## Production build

Create an optimized bundle in `dist/`:

```bash
npm install
npm run build
```

Preview that build without deploying:

```bash
npm run preview
```

Then open the URL shown in the terminal (often `http://localhost:4173`).
